HR Overtime Management

A comprehensive solution for managing employee overtime hours and payments in Odoo.

Key Features

Multiple Overtime Types

  • Regular Overtime (1.5x rate)
  • Weekend Overtime (2.0x rate)
  • Holiday Overtime (2.5x rate)
  • Night Shift Overtime (1.75x rate)
  • Emergency Overtime (2.0x rate)
  • Create custom overtime types with configurable rates

Approval Workflow

  • Draft → Submitted → Approved/Rejected workflow
  • Manager notifications for pending approvals
  • Activity scheduling for better tracking
  • Rejection reason tracking

Automatic Calculations

  • Automatic duration calculation from start/end times
  • Hourly rate computed from employee contract
  • Overtime amount based on type multiplier
  • Configurable monthly working hours

Reporting & Analytics

  • Pivot and Graph views for analysis
  • Calendar view for visual tracking
  • Report wizard with filtering options
  • Export capabilities

Security & Access

  • User and Manager permission groups
  • Employees see only their own records
  • Managers see their team's records
  • HR Managers have full access
  • Multi-company support

Configuration

Navigate to Settings → HR → Overtime Settings to configure:

  • Monthly working hours (default: 173.33 hours)
  • Maximum overtime hours per request (default: 12 hours)
  • Approval requirements

How to Use

  1. Install the module from Apps menu
  2. Configure overtime types in Configuration → Overtime Types
  3. Employees create overtime requests from Overtime → My Overtime
  4. Managers approve/reject from Overtime → To Approve
  5. Generate reports from Overtime → Reports
Note: This module requires the HR and HR Contracts modules to be installed.
